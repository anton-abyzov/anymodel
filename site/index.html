How It Works

Pick your setup. Both use your own key.

Get a free OpenRouter API key (no credit card for free models). Your key — your models — your machine.

A

Remote Proxy — simplest

One command. No npm, no install, no proxy to run. Just set your OpenRouter key and go.

ANTHROPIC_BASE_URL=https://api.anymodel.dev ANTHROPIC_API_KEY=sk-or-v1-your-key claude

The remote proxy at api.anymodel.dev uses YOUR key to call OpenRouter. Nothing stored server-side.

B

Local Proxy — full control

Run the proxy on your machine. Two terminals. Everything stays local.

OPENROUTER_API_KEY=sk-or-v1-... npx anymodel proxy

Then in another terminal:

ANTHROPIC_BASE_URL=http://localhost:9090 claude

Pick a model: npx anymodel proxy --model qwen/qwen3-coder:free

Both work with your existing claude install. No cloning needed. Free models cost $0.

CLI Reference

anymodel [provider] [options]

Providers:
  openrouter    Route through OpenRouter
  ollama        Route through local Ollama
  remote        Free-only + auth (for deployment)

Options:
  --model, -m     Model ID (e.g., qwen/qwen3-coder:free)
  --port, -p      Proxy port (default: 9090)
  --free-only     Only allow free models
  --token, -t     Require auth token
  --rpm           Rate limit per minute (default: 60)
  --help, -h      Show help

Environment Variables

OPENROUTER_API_KEY Your OpenRouter API key
OPENROUTER_MODEL Default model override
PROXY_PORT Proxy listen port (default: 9090)
ANYMODEL_TOKEN Auth token for proxy access
ANYMODEL_REMOTE_URL Remote proxy URL override

anymodel auto-loads .env from the current directory.

Health Endpoint

GET /health

Response:
{
  "status": "ok",
  "version": "1.0.0",
  "provider": "openrouter",
  "model": "google/gemini-2.5-flash",
  "uptime": 3600.5,
  "timestamp": "2025-01-15T10:30:00Z"
}

Request Routing

/v1/messages → Routed to your chosen provider
/v1/* → Passed through to api.anthropic.com
/health → Returns proxy status JSON

The proxy sanitizes request bodies: strips betas, metadata, thinking, cache_control, and normalizes tool_choice for cross-provider compatibility.

Examples

Real-world usage patterns.

Copy-paste examples for common scenarios.

Remote

Use free models instantly

# No install. No API key. Just run Claude Code:
ANTHROPIC_BASE_URL=https://api.anymodel.dev \
ANTHROPIC_API_KEY=anymodel-remote-token-2026 \
claude

# That's it. Free models, zero setup.
Claude Code

Local proxy with a specific model

npx anymodel --model google/gemini-2.5-flash

# In another terminal, launch Claude Code
ANTHROPIC_BASE_URL=http://localhost:9090 claude

npx anymodel --model anthropic/claude-sonnet-4.6

Remote

Remote proxy for teams

# Deploy as a shared proxy (free models, auth, rate limit)
npx anymodel remote --token my-team-secret

# Team members connect from any machine:
ANTHROPIC_BASE_URL=http://your-server:9090 \
ANTHROPIC_API_KEY=my-team-secret \
claude

# Defaults: free-only, 60 req/min, token required.
Ollama

Run locally with Ollama

# Start Ollama
ollama serve

# Pull a model
ollama pull llama3

# Start anymodel with Ollama
npx anymodel ollama --model llama3

# Fully offline. Nothing leaves your machine.

Get Started

No install needed. Or install locally.

Use the hosted remote proxy instantly, or run locally with npx.

Remote proxy (no install):

$ ANTHROPIC_BASE_URL=https://api.anymodel.dev claude
or run locally:
$ npx anymodel
or install globally:
$ npm i -g anymodel
